# Конспект лекцій: Архітектура та апаратне забезпечення ПК

## 1. Правила електробезпеки при роботі з внутрішніми компонентами ПК

Перш ніж відкривати корпус комп'ютера, критично важливо дотримуватися правил безпеки, щоб уникнути ураження електричним струмом та пошкодження компонентів статичною електрикою.

**Основні кроки:**

1.  **Повне знеструмлення:**
    * Вимкніть комп'ютер через операційну систему.
    * Перемкніть тумблер на блоці живлення (ззаду корпусу) у положення "0" (Off).
    * **Обов'язково** вийміть кабель живлення з розетки та з блоку живлення.
2.  **Зняття залишкового заряду:**
    * Після відключення кабелю натисніть і утримуйте кнопку увімкнення на корпусі протягом 10–15 секунд. Це розрядить конденсатори на материнській платі та в блоці живлення.
3.  **Захист від електростатичного розряду (ESD):**
    * Статична електрика може миттєво знищити мікросхеми.
    * Використовуйте антистатичний браслет, підключений до заземленого предмета.
    * Якщо браслета немає, періодично торкайтеся нефарбованої металевої частини корпусу (наприклад, шасі) або батареї опалення, щоб «заземлити» себе.
4.  **Умови роботи:**
    * Працюйте на чистій, сухій поверхні (бажано не на килимі, який генерує статику).
    * Не торкайтеся контактів ("золотих пальців") на платах оперативної пам'яті чи відеокарти руками.

## 2. Форм-фактори корпусів настільних комп'ютерів

Форм-фактор корпусу визначає його розмір, кількість внутрішнього простору та тип материнської плати, яку можна встановити.

| Тип корпусу | Опис | Сумісність з платами |
| :--- | :--- | :--- |
| **Full Tower** | Найбільший тип. Забезпечує відмінну вентиляцію, багато місця для дисків та водяного охолодження. | E-ATX, ATX, Micro-ATX |
| **Mid Tower** | Найпопулярніший стандарт. Баланс між розміром та місткістю. | ATX, Micro-ATX, Mini-ITX |
| **Mini Tower** | Компактні корпуси, зазвичай мають менше слотів розширення та гіршу вентиляцію. | Micro-ATX, Mini-ITX |
| **SFF (Small Form Factor)** | Надкомпактні корпуси для медіацентрів або офісів. Вимагають специфічних компонентів (низькопрофільні відеокарти). | Mini-ITX |

## 3. Блок живлення (PSU)

Блок живлення (БЖ) перетворює змінний струм (AC) з розетки на постійний струм (DC), необхідний для компонентів ПК.

### Основні характеристики:
* **Потужність (Ват, W):** Визначає, скільки енергії БЖ може віддати компонентам (наприклад, 500W, 750W, 1000W). Розраховується як сума споживання всіх компонентів + 20-30% запасу.
* **Сертифікат 80 PLUS:** Показник коефіцієнта корисної дії (ККД).
    * *Рівні:* White, Bronze, Silver, Gold, Platinum, Titanium. Чим вищий рівень, тим менше енергії втрачається на нагрів.
* **Модульність:**
    * *Non-modular:* Усі кабелі впаяні в блок.
    * *Semi-modular:* Основні кабелі впаяні, периферійні можна від'єднати.
    * *Full-modular:* Усі кабелі знімні (зручно для кабель-менеджменту).

### Конектори блоку живлення:

1.  **24-pin ATX:** Основне живлення материнської плати.
2.  **CPU 4+4 pin (EPS):** Живлення процесора.
3.  **PCIe 6+2 pin:** Додаткове живлення для відеокарт.
4.  **SATA Power:** Живлення жорстких дисків (HDD), SSD та контролерів підсвітки.
5.  **Molex:** Застарілий роз'єм, використовується для старих дисків або корпусних вентиляторів.

## 4. Основні компоненти материнської плати (7 елементів)

Материнська плата — це "хребет" комп'ютера, що з'єднує всі компоненти.

1.  **Сокет процесора (CPU Socket):**
    Роз'єм, у який встановлюється процесор. Має бути сумісним з типом процесора (наприклад, AM4/AM5 для AMD або LGA1700/1851 для Intel).
2.  **Чіпсет (Chipset):**
    Набір мікросхем, що керує передачею даних між процесором, пам'яттю та периферією. Визначає можливості плати (розгін, кількість USB-портів).
3.  **Слоти оперативної пам'яті (DIMM Slots):**
    Роз'єми для встановлення модулів RAM (DDR4 або DDR5). Зазвичай їх 2 або 4.
4.  **Слоти розширення (PCIe Slots):**
    Використовуються для встановлення відеокарт, звукових карт, мережевих адаптерів. Основний слот — x16 (для відеокарти).
5.  **VRM (Voltage Regulator Module):**
    Система живлення навколо сокета. Вона перетворює 12В від блоку живлення у низьку напругу (близько 1.0–1.4В) для процесора. Якість VRM важлива для стабільності потужних CPU.
6.  **Роз'єми для накопичувачів (SATA та M.2):**
    * *SATA:* Для підключення класичних HDD та 2.5" SSD.
    * *M.2:* Слот для встановлення швидкісних NVMe SSD прямо на плату.
7.  **Задня панель вводі/виводу (I/O Panel):**
    Блок роз'ємів, що виходить на задню частину корпусу (USB порти, аудіороз'єми, LAN-порт, HDMI/DisplayPort).
